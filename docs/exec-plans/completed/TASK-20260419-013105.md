# fix: TASK-20260419-013105 InquiryMessageJpaEntity @Lob 제거 및 columnDefinition TEXT 추가

## 배경 및 목표

Hibernate 6에서 `@Lob` + String 타입은 MySQL에서 `tinytext` 컬럼으로 매핑되는데, 
실제 DB 스키마는 `TEXT` 컬럼으로 정의되어 있어 schema validation이 실패한다.

**목표**: 
- `InquiryMessageJpaEntity`의 `body` 필드에서 `@Lob` 제거
- `@Column` 정의에 `columnDefinition = "TEXT"` 추가
- Unused import 제거
