## 톤(Tone) 3축 포지셔닝

### 축 1: 전문성 ↔ 친근성

```
전문적 ←───────────────────→ 친근함
  ↑                            ↑
우리                          우리
```

**Tourwave의 포지션: 70% 전문적 + 30% 친근함**

- **전문성:** 강사 자격, 안전 정책, 환불 규칙 등은 명확하고 정확하게
- **친근함:** 고객과의 소통은 편하고 따뜻하게, 너무 딱딱하지 않게
- **예시:**
  - ❌ "예약 취소 규정을 숙지하시기 바랍니다." (너무 형식적)
  - ✓ "7일 전에 취소하면 전액 환불됩니다. 걱정 없어요." (정확 + 따뜻함)

---

### 축 2: 진지함 ↔ 재미

```
진지함 ←───────────────────→ 재미
 ↑                           ↑
우리                        우리
```

**Tourwave의 포지션: 60% 진지함 + 40% 재미**

- **진지함:** 안전, 신뢰, 데이터 투명성 — 절대 장난스럽게 다루지 않음
- **재미:** 고객의 경험과 모험심을 축하하고, 감정적으로 연결
- **예시:**
  - ❌ "하늘을 날아보세요 ㅋㅋ" (너무 가벼움)
  - ✓ "하늘을 나는 설렘, Tourwave가 안전하게 지켜드립니다." (진지함 + 감정)

---

### 축 3: 보수적 ↔ 대담함

```
보수적 ←───────────────────→ 대담함
 ↑                           ↑
우리
```

**Tourwave의 포지션: 65% 보수적 + 35% 대담함**

- **보수적:** 고객 데이터, 환불, 취소 정책 — 약속은 지키고, 기준은 명확
- **대담함:** 새로운 기능(패키지, 단체 할인), 시장 도전 메시지는 자신있게
- **예시:**
  - ❌ "환불은... 음, 경우에 따라 다릅니다." (애매함)
  - ✓ "7일 전 취소는 100% 환불. 명확하고 공정합니다." (명확함)

## 톤 특성 정의

### 전체 톤의 5대 특성

| 특성 | 설명 | 예시 |
|------|------|------|
| **투명함** | 규정, 가격, 환불을 명확히 설명. 숨겨진 것 없음 | "수수료 3%만 있고, 나머지는 모두 운영사에게 갑니다." |
| **신뢰성** | 약속은 지킨다. 데이터와 증거로 말한다 | "99.8%의 강사가 인증받은 자격증을 보유했습니다." |
| **따뜻함** | 고객의 불안감을 인정하고 격려한다 | "처음 해보는 거라 불안하신가요? 그럴 수 있어요. 우리가 함께 하겠습니다." |
| **역량감** | "이 문제는 해결할 수 있다"는 자신감 | "예약 시스템이 복잡해 보여도, 3분이면 완료돼요." |
| **정직함** | 좋은 점만 말하지 않고, 제약도 설명 | "비용이 싸지는 않습니다. 대신 안전과 투명성으로 돈값을 하겠습니다." |

## 금지 단어 & 표현

### 절대 쓰지 않을 표현

| 표현 | 이유 | 대체 표현 |
|------|------|---------|
| "따라오세요" | 상명하복 느낌, 친근성 떨어짐 | "함께 하세요", "가볼까요" |
| "규정상", "할 수 없습니다" | 거부적, 딱딱함 | "안전하게 하려면", "이렇게 해드립니다" |
| "강력하다", "최고다" | 입증 없는 자랑, 신뢰도 떨어짐 | 구체적 수치나 고객 리뷰로 증명 |
| "쉽다", "간단하다" | 주관적, 모든 사용자에게 다름 | "3분이면 완료돼요", "이 부분은 우리가 처리해요" |
| "여행객들이 원하는", "모두가 좋아한다" | 일반화, 편견 | "많은 고객이", "여러 후기에서" (구체적 근거) |
| "언제어디서나", "항상" | 과장, 입증 불가 | "제주, 강원에서", "주말·휴가 때" |
| "대박", "핵", "꿀팁" | 너무 캐주얼, 브랜드 신뢰도 손상 | "추천해요", "도움이 돼요", "알아두면 좋은" |

## 청중별 톤 규칙

### 1. 운영사(B2B) 대상 톤

**기본 포지션:** 70% 전문적 + 80% 진지함 + 90% 투명성

**특징:**
- 비즈니스 용어, 수치, ROI 중심
- 시간 절약, 효율성을 강조
- 약속은 명확히, 기술적 세부사항도 설명
- 파트너 관계 강조 (고객이 아닌 사업 파트너)

**톤 가이드:**

| 상황 | 좋은 예 | 나쁜 예 |
|------|--------|--------|
| 온보딩 초기 | "월 15시간의 관리 시간을 절약할 수 있습니다. 데이터로 증명합니다." | "쉬워요!" |
| 기술 지원 | "API 연동이 필요하신가요? 개발팀에서 3일 내에 지원하겠습니다." | "언제든지 도와드려요." |
| 문제 발생 | "결제 동기화 이슈를 파악했습니다. 24시간 내에 해결하겠습니다." | "미안합니다." |
| 새 기능 제안 | "단체 할인 자동화로 예상 GMV 20% 증가. 이달 말에 출시됩니다." | "좋은 기능을 준비하고 있어요!" |
| 월별 리포트 | "지난달 환불율 2%, 고객 만족도 4.7/5.0. 목표 달성했습니다." | "좋은 결과가 나왔어요." |

**금지 표현 (B2B):**
- "재미있게 시작해보세요" — 비즈니스 문맥에서 너무 캐주얼
- "당신도 할 수 있어요" — 능력을 의심하는 톤
- "최신 기술을 사용합니다" — 구체성 부족

---

### 2. 소비자(B2C) 대상 톤

**기본 포지션:** 60% 전문적 + 40% 재미 + 60% 따뜻함

**특징:**
- 감정과 경험 중심 (스펙이 아니라 "느낌")
- 불안을 인정하고 안심시키기
- 친근한 말투, 하지만 신뢰성은 확보
- 다음 단계를 명확하게 안내

**톤 가이드:**

| 상황 | 좋은 예 | 나쁜 예 |
|------|--------|--------|
| 첫 방문 | "처음 해보는 거 맞죠? 리뷰 많은 강사부터 추천해드릴게요." | "추천 상품을 누르세요." |
| 예약 확인 | "예약이 완료됐어요. 강사 연락처는 여기, 취소는 언제든지 OK." | "예약 번호: 12345" |
| 취소 요청 | "7일 전이니까 전액 환불이에요. 3일 내에 입금될 거고, 문자로 알려드릴게요." | "환불 규정을 숙지하시기 바랍니다." |
| 나쁜 리뷰 대응 | "안타까운 경험을 하셨네요. 직접 연락해서 해결책을 찾겠습니다." | "의견 감사합니다." |
| 활동 후 | "정말 즐거웠죠? 리뷰 한두 줄만 남겨주면, 다른 사람들도 신뢰하고 올 거예요." | "리뷰를 작성해주세요." |
| 환불 실패 | "환불이 늦어져서 미안해요. 지금 추적 번호를 보내드릴 테니, 언제든지 물어봐도 괜찮아." | "환불을 조회 중입니다." |

**금지 표현 (B2C):**
- "규정상 불가능합니다" — 거부적, 도움이 안 되는 느낌
- "다시 시도해주세요" — 사용자 탓하는 톤
- "전문가 수준" — 입문자를 배제하는 표현

---

### 3. 마케팅/광고 대상 톤

**기본 포지션:** 50% 전문적 + 50% 재미 + 70% 대담함

**특징:**
- 감정적 공감 + 명확한 이점
- 스토리와 데이터 혼합
- 행동 유도 명확
- 브랜드 신뢰도를 높이되, 재미 요소도 포함

**톤 가이드:**

| 채널 | 좋은 예 | 나쁜 예 |
|------|--------|--------|
| 인스타그램 | "하늘을 나는 기쁨, 안전한 손길로. 리뷰 4,000+ ⭐4.8" | "레저 스포츠를 즐기세요!" |
| 네이버 검색광고 | "환불 걱정 없이 예약하세요 — 7일 전 100% 환불 보장" | "최고의 액티비티 플랫폼" |
| 유튜브 프리롤 | "첫 시도는 불안할 수 있어요. 2만 명이 이미 경험했고, 4.8점을 남겼어요." | "지금 예약하세요!" |
| 이메일 뉴스레터 | "[제주] 카약 경험담: 초보자의 눈으로 본 3시간. 가는 길부터 돌아오는 길까지." | "새로운 액티비티가 추가되었습니다." |

## 조건부 톤 조정

### 위기 상황 (환불 미처리, 강사 사고 등)
- **추가 규칙:** 투명성 +40%, 공감 +50%
- **금지:** 변명, 회피, 지연 표현
- **필수:** 사실 인정 → 즉시 조치 → 예방안 제시

**예시:**
```
❌ "죄송한 일이 발생했습니다. 조사 후 답변드리겠습니다."
✓ "환불이 예약된 날짜에 처리되지 않았습니다. 저희 실수입니다. 
   지금 바로 수동 처리하고, 3일 내에 완료될 예정입니다. 
   추가로 고객 마일리지도 드리겠습니다."
```

### 거절/제한 상황
- **추가 규칙:** 따뜻함 +30%, 대안 제시 필수
- **금지:** "불가능", "안 됩니다"
- **필수:** 이유 설명 → 대안 제시 → 다음 단계

**예시:**
```
❌ "20명 이상 단체는 처리할 수 없습니다."
✓ "20명 이상 단체는 별도 담당팀에서 맞춤 지원해드립니다. 
   contact@tourwave.com로 연락 주시면, 
   2시간 내에 전담자가 최적의 가격을 제안해드릴게요."
```
